# Reagent Leaflet #

Simple ClojureScript example for using LeafletJS maps as Reagent components.


Demo at http://webjure.org/reagent-leaflet/ (see example.cljs for the code).


## Running locally ##

Clone this repo and run: clojure -M:fig:build
